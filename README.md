AI Powered Crime Prevention Platform

## Problem
Many crime related incidents go unreported due to fear, lack of awareness and
absence of simple reporting platforms.

## Solution
SafeSphere is an AI powered web platform where users can report incidents.
Google Gemini AI analyzes the input and classifies the crime type.

## Features
- Crime reporting
- AI based classification
- Secure and anonymous access
- Cloud based storage

## Google Technologies Used
- Google Gemini API
- Firebase
- Google Cloud Platform

## Tech Stack

- Google Gemini
